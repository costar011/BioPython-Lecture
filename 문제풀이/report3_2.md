# 📋 주민등록번호와 전화번호 유효성 검사 프로그램 만들기

## 💡 문제 설명

함수 `check_private_data(social_number, number)`가 주어집니다.
해당 함수는 **주민등록번호**와 **전화번호**가 모두 유효한 경우 **"정상"**을 출력하고, 하나라도 유효하지 않다면 **"오류"**를 출력합니다.

---

## 🔒 주민등록번호 (`social_number`) 유효성 검사 항목

주민등록번호는 **YYMMDD-GHIJKLX** 형식으로 구분합니다.

### 조건 1: 형식 및 문자 구성

주민등록번호는 **하이픈(`-`)**을 제외한 모든 부분이 **숫자**로 이루어져야 합니다. 그렇지 않은 경우 오류입니다.

- **(정상 예)** `881016-1234567`
- **(오류 예)** `88101A-1234567`

### 조건 2: 길이

주민등록번호의 총 길이는 **14자리**입니다. (하이픈 포함)

### 조건 3: 생년월일 유효성 (미래 날짜 불가)

생년월일(YYMMDD)은 프로그램 실행 날짜를 기준으로 **미래 날짜일 수 없습니다.**

|    구분    | 성별 코드 (G) | 출생 연도 |            성별            |
| :--------: | :-----------: | :-------: | :------------------------: |
| 1900년대생 |   1 또는 2    | **19**YY  | 남자(홀수) 또는 여자(짝수) |
| 2000년대생 |   3 또는 4    | **20**YY  | 남자(홀수) 또는 여자(짝수) |

- **예시 1:** `881016-1234567` $\rightarrow$ 1988년 10월 16일생, 남자 (정상)
- **예시 2:** `881016-3123456` $\rightarrow$ 2088년 10월 16일생. 프로그램 실행 날짜 이후이므로 **오류**에 해당합니다.

## 📞 전화번호 (`number`) 유효성 검사 항목

전화번호는 **0A(B)-CDE(F)-GHIJ** 형식으로, 괄호로 묶인 (B) 와 (F)는 생략 가능합니다.

### 조건 1: 지역번호 유효성

전화번호는 **0으로 시작**하며, 입력한 지역번호가 아래 목록에 존재해야 합니다.

| 지역번호 | 관할 지역                              |
| :------: | :------------------------------------- |
|  **02**  | 서울특별시, 과천시, 광명시             |
| **031**  | 경기도                                 |
| **032**  | 인천광역시, 부천시                     |
| **033**  | 강원도                                 |
| **041**  | 충청남도                               |
| **042**  | 대전광역시, 계룡시, 공주시 반포면 일부 |
| **043**  | 충청북도                               |
| **044**  | 세종특별자치시                         |
| **051**  | 부산광역시                             |
| **052**  | 울산광역시                             |
| **053**  | 대구광역시, 경산시                     |
| **054**  | 경상북도, 대구광역시 군위군            |
| **055**  | 경상남도                               |
| **061**  | 전라남도                               |
| **062**  | 광주광역시                             |
| **063**  | 전북특별자치도                         |
| **064**  | 제주특별자치도                         |

### 조건 2: 가입자 번호 길이

가입자번호는 하이픈 기준으로 다음과 같이 길이를 만족해야 합니다.

- 첫 번째 그룹 (CDE(F)): **3~4자리**
- 두 번째 그룹 (GHIJ): **4자리**

- **정상 예:** `02-123-1234` 또는 `031-1234-1234`
- **오류 예:** `02-12-1234` (3자리 미만) 또는 `031-1234-12` (4자리 미만)
